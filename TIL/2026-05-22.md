## 배운 것
- **react-hook-form `reset()` 동작**: `reset(undefined)`는 아무 동작 안 함. 인자 없이 `reset()`을 호출하면 `useForm` 초기화 시점의 `defaultValues`로 리셋됨. 날짜처럼 동적 값이 필요하면 별도 함수로 추출하여 `reset(getDefaultValues())` 형태로 호출해야 현재 시점 기준으로 초기화 가능
- **AppMenu 역할 기반 필터링 전략**: 기존 `filteredModel`의 복잡한 역할별 분기를 건드리지 않고, 마지막 단계에서 `finalModel`로 한 번 더 필터링하는 방식이 기존 코드 영향 없이 안전하게 메뉴를 추가/제거할 수 있음

## 이슈 / 트러블슈팅
- **`git rm --cached` 후 `git reset` 주의**: `git rm --cached`로 캐시 제거 후 실수로 `git reset HEAD -- .`를 하면 제거가 되돌려짐. 캐시 제거와 스테이징은 한 흐름에서 처리해야 함
